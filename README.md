# Weather Forecast Application

A responsive weather forecast web application built using HTML, Tailwind CSS, and JavaScript. 
The application allows users to view current weather conditions and extended forecasts 
based on their current location or searched city.

## Features

- Search weather by city name
- Get weather for current location using Geolocation API
- Displays temperature, humidity, wind speed, and weather condition
- 5-day extended weather forecast
- Temperature unit toggle (°C / °F) for today’s temperature
- Recently searched cities dropdown
- Dynamic background based on weather condition (rainy, cloudy, clear, etc.)
- Custom error messages for invalid cities and API errors
- Responsive design for all device, mobile, tablet, and desktop screens etc.

## Tech Stack

- HTML5
- Tailwind CSS(CLI)
- JavaScript
- OpenWeatherMap API

## API Used

- OpenWeatherMap API
  - Current Weather API
  - Next 5 Days forecasting

Weather data is fetched using latitude & longitude for accurate location-based results.

## How to Run the Project

1. Clone the repository
2. Open the project folder
3. Open `index.html` in any browser
4. Allow location access to view current/your weather
